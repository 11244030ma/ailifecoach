## Typography Scale

```
12px  Small text (timestamps, meta)
14px  Secondary text (buttons, labels)
16px  Body text (messages, input)
18px  Emphasized text (header subtitle)
20px  Large text (header title)
24px  Heading (welcome title)
```

## Animation Timing

```
Fast:     0.1s  (button press)
Normal:   0.2s  (hover effects, transitions)
Slow:     0.3s  (message fade-in, modal)
Smooth:   0.4s  (typing dots delay)
```

## Interaction States

### Input Field
```
Empty:       [Type your message...        ]  Gray bg
Focused:     [Type your message...        ]  White bg, blue ring
Typing:      [Hello, I need help with...  ]  White bg, blue ring
Disabled:    [Type your message...        ]  Gray bg, no interaction
```

### Message States
```
Sending:     [Message content...] ⟳         Slightly faded
Sent:        [Message content...]           Full opacity
Error:       [Message content...] ⚠         Red indicator
```

## Quick Start Commands

### React + TypeScript
```bash
# Install dependencies
npm install react react-dom typescript @types/react @types/react-dom

# Copy component files
cp ui-components/ChatInterface.tsx src/
cp ui-components/ChatInterface.css src/

# Import in your app
import ChatContainer from './ChatInterface';
```

### Tailwind CSS Alternative
```bash
# Install Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Use utility classes instead of custom CSS
# See worklife-chat-ui-design.md for class mappings
```

### Figma Setup
```
1. Create 3 frames: 375px, 768px, 1440px
2. Import color styles from palette
3. Set up text styles (12px-24px scale)
4. Create components: Message, Button, Input
5. Use auto-layout for responsive behavior
```
