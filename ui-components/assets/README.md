# Assets Folder

## Logo

Place your logo file here as `logo.png` (or `logo.svg`)

The logo should be:
- Square format (recommended: 512x512px or larger)
- Transparent background (PNG with alpha channel)
- Navy blue (#1A2A40) as primary color
- High resolution for retina displays

### Current Logo Usage:
- Landing page hero section
- Authentication pages
- Chat interface header

### To Update Logo:
1. Save your logo file as `logo.png` in this folder
2. Refresh the browser
3. The logo will automatically appear on all pages

### Supported Formats:
- PNG (recommended for transparency)
- SVG (best for scalability)
- JPG (if no transparency needed)
